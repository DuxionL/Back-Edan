### tutor cara pake method2 di students
1. POST /api/students - Create New Student Biodata
- `localhost:5000/api/students`
- Body: 
- { "name": "String",
  "address": "String",
  "phoneNumber": "String" } 
- name itu wajib, address sama phoneNumber optional
- habis post, akan return variabel studentId autogenerated, 9 digit integer (sistem NIM belom implementasi)
2. GET /api/students - View All Students
- `localhost:5000/api/students`
3. GET /api/students/{id} - Read Student Biodata by ID
- `localhost:5000/api/students/{studentId}`
- mengambil data student dengan Id yang ditentukan
4. PUT /api/students/{studentId} - Update Student Biodata
- `localhost:5000/api/students/{studentId}`
- Body:
- { "address": "Updated Address: String",
  "phoneNumber": "String" }
- !!! cuman address sama phoneNumber bisa diganti, satu atau 22nya sekaligus
- (yah masa ganti identitas)
5. DELETE /api/students/{studentId} - Delete Student Data
- `localhost:5000/api/students/{studentId}`
- ilangin anak dari databes, masukin ke backrooms
